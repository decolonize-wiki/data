# decolonize.wiki — open dataset

Published analyses from [decolonize.wiki](https://decolonize.wiki), one JSON
file per analysis, plus the pinned plain-text article extracts they were
verified against.

- `analyses/<lang>/<slug>/<sequence>.json` — e.g. `analyses/en/brazil/1.json`
- `extracts/<lang>/<slug>/<revisionId>.txt` — the exact text analyzed

Every analysis records its Wikipedia revision ID and methodology version
(see the [methodology repo](https://github.com/decolonize-wiki/methodology)).
Article text is from Wikipedia, CC BY-SA. This dataset: CC BY-SA 4.0.
Contains no personal data by construction.
